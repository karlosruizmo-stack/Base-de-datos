# Base-de-datos
Este programa implementa un sistema de verificación de usuarios para una base de datos simulada.
Cada usuario está asociado a un número identificador, y la verificación de acceso se realiza mediante una contraseña vinculada a dicho número.

El sistema permite un número limitado de intentos y controla posibles errores mediante el uso de try-catch, evitando que el programa se cierre de forma inesperada.
